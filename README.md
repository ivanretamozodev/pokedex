# Development

1. Clone repository
2. type

```
npm install
```

3. need Nest CLI installed

```
npm i -g @nestjs/cli
```

4. run database

```
docker-compose up -d
```

5. rebuild data db with seed

```
http://localhost:3000/api/v1/seed
```
